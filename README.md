# My Number with Redux
> 리덕스 공부용 프로젝트

## 작동  
- 이름과 숫자를 입력받습니다.
- 더하기 버튼을 누르면 입력받은 숫자에 더합니다.

## 실행
[Redux DevTools](https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd?hl=en) 설치  
```shell
yarn install
yarn start
```

## 사용  
- React
- CRA
- typescript
- Redux
- Redux-Persist
- Emotion
